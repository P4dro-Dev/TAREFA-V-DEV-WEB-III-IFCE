# TAREFA-V-DEV-WEB-III-IFCE
O projeto é uma aplicação Django que gerencia autores usando ORM, permitindo cadastrar, listar, buscar, editar e excluir registros. Inclui modelo de dados, CRUD completo, templates organizados, integração com o admin e versionamento seguindo boas práticas com criação e merge da branch banco-de-dados.
